# **App Name**: FinTrack

## Core Features:

- Expense Tracking: Automatically track expenses from linked bank accounts and credit cards.
- Spending Categorization: Automatically categorize expenses (e.g., groceries, transportation) using a transaction classifier LLM tool.
- Budget Setting: Allow users to set monthly budgets for each spending category.
- Visualized Spending: Present spending data through interactive charts (bar, pie, line) to visualize spending patterns over time.
- Dashboard Insights: Provide a dashboard summarizing key insights like total spending, budget status, and top spending categories.
- Goal Setting: Enable users to set financial goals and track progress towards them.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to represent trust and financial stability.
- Background color: Very light blue (#F0F8FF) for a clean and calming interface.
- Accent color: Soft green (#90EE90) to indicate positive trends and budget surpluses.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look suitable for both headlines or body text
- Use flat, minimalist icons for categories and actions.
- Emphasize a clean, card-based layout for clear information presentation.
- Subtle transitions when navigating between views.
